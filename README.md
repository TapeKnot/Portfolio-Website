This is my portfolio website, created using Next.js.

## View Online

My website is currently hosted on Vercel and can be found at [https://www.benjaminlevy.dev/](https://www.benjaminlevy.dev/).

## Download & Install

Before starting, you must install Git and Node.js onto your machine.

First, fork and clone this repository. Open the terminal in the repository directory and run ```npm start``` to run a development build.

Open [http://localhost:3000](http://localhost:3000) in your browser to see my website!
